# Catálogo de Monitoração FinOps - Darede

Este projeto apresenta uma página de comparativo de planos FinOps da Darede, desenvolvida para demonstrar os diferentes níveis de serviço oferecidos para otimização de custos na nuvem.

## 📋 Sobre o Projeto

A página exibe uma tabela comparativa elegante e responsiva que apresenta três planos de serviços FinOps:

- **FinOps Essential**: Plano básico com funcionalidades fundamentais
- **FinOps Pro**: Plano intermediário com recursos avançados
- **FinOps Ultimate**: Plano premium (em desenvolvimento)

### Características da Página

- **Design Responsivo**: Adaptável para desktop, tablet e mobile
- **Interface Intuitiva**: Tooltips informativos para cada funcionalidade
- **Paleta de Cores Corporativa**: Utiliza as cores da marca Darede
- **Organização por Fases**: Recursos organizados em Onboarding, Foundation e Operation
- **Visual Discreto para Ultimate**: Plano em desenvolvimento apresentado de forma sutil

## 🚀 Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com Flexbox e Grid
- JavaScript vanilla para interatividade
- Design responsivo com media queries
- Gradientes e animações CSS

## 🎨 Paleta de Cores

A página é totalmente responsiva com breakpoints para:

- Tablet (768px - 1199px)

## 🛠️ Como Executar

### Desenvolvimento Local

```bash
# Instalar dependências
npm install

# Executar servidor de desenvolvimento
npm run dev
```

O servidor será iniciado na porta 3000: `http://localhost:3000`

### Deploy no Bolt

Este projeto foi desenvolvido no Bolt.new e pode ser facilmente deployado:

1. **Deploy Automático**: Use o botão de deploy no Bolt para publicar automaticamente
2. **Netlify**: Compatível com deploy direto no Netlify
3. **Servidor Estático**: Pode ser servido por qualquer servidor de arquivos estáticos

## 📊 Funcionalidades da Tabela

### Fases dos Serviços

1. **Onboarding**
   - Entendimento da Cloud
   - Estrutura de Contas
   - Consumo Mensal
   - Política de Tags

2. **Foundation**
   - Setup de Alarmes
   - Assess Maturidade
   - Dashboards Customizados
   - Automações
   - Tagueamento

3. **Operation**
   - Suporte Contínuo
   - Recomendações
   - Apresentações Executivas
   - Webinars
   - Automações Avançadas

### Recursos Interativos

- **Tooltips**: Informações detalhadas sobre cada funcionalidade
- **Hover Effects**: Feedback visual ao passar o mouse
- **Sticky Headers**: Cabeçalhos fixos durante o scroll
- **Animações Suaves**: Transições CSS para melhor UX

## 🎯 Objetivos do Projeto

- Apresentar claramente os diferentes planos FinOps
- Facilitar a comparação entre funcionalidades
- Demonstrar a progressão de valor entre os planos
- Manter o foco nos planos disponíveis (Essential e Pro)
- Preparar terreno para o lançamento futuro do Ultimate
